# BackendPractice
#
# I created this folder to practice backend development with Express, Mongoose, and data modeling.
#
# Details:
# - Purpose: Learn and experiment with building REST APIs using Express.js.
# - Database: Use MongoDB with Mongoose for schema design and data modeling.
# - Topics covered: routing, middleware, validation, authentication (optional), relations and schema design, and testing endpoints.
#
# How to use:
# 1. Install dependencies: npm install
# 2. Start the server: npm run dev (or node index.js)
# 3. Connect to MongoDB (configure connection string in .env)
